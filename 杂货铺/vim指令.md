# VIM指令

## 1. 剪切

```shell
# 快捷方式操作
dd:删除并剪切当前行
n + dd:删除并剪切从当前行开始的n行
将光标移动至目标行，按p粘贴(p粘贴到当前行的下一行，P可以粘贴到当前行的上一行)
# 命令行方式
:1,10 m 12    # 将1至10行内容剪切到12行
```

## 2. 复制

```shell
# 快捷方式操作
yy:复制当前行
n + yy:复制从当前行开始的n行
将光标移动至目标行，按p粘贴(p粘贴到当前行的下一行，P可以粘贴到当前行的上一行)
# 命令行方式
:1,10 co 12    # 将1至10行内容复制到12行
```
